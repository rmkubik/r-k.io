Island Maker Patch Notes

Play the game!

I have listed patch notes on this site because they contain spoilers. Itch.io devlogs don't seem to have a spoiler option. If you don't wish to see spoilers, do not expand the sections below!

Version 7 - 3/12/22

  • An object was missing from the Journal on accident. The "rocky seed" object was added to the journal.
  • The object "lighthouse" had its functionality changed slightly. It used to provide 2 ships every time it was placed. It now provides one ship per adjacent ocean tile.
  • The object "quarry" had its functionality changed slightly. It used to randomly give an object from a set of options with even distribution. They were Rocky Seed, Mine, and Dungeon. It will now provide the same items. But, new rules effect the distribution chances conditionally. Each adjacent mountain is a chance for a "Rocky Seed". Each adjacent mine is a chance for a "Mine" or "Dungeon".
